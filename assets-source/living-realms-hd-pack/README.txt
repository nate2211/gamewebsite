Voxel Frontier Living Realms HD Source Pack

Six original 4096x4096 editable texture atlases. Each atlas contains 64 material tiles with layered macro grain, micro grain, seams, rivets, straight fractures, and voxel-safe edge detail. These source images are included for future runtime atlas baking and custom model authoring; the shipping game uses the new selectable 32-128 px procedural texture pipeline to avoid loading multi-hundred-megabyte source art into browser memory.
