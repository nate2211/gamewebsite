# Voxel UI Pack

`src/assets/ui-voxel/` and `public/assets/ui-voxel/` contain 40 original 1024×1024 pixel-block surfaces used by the inventory, workbench, furnace, armor, stats, perks, quests, codex, performance, backups, pause, item inspection, and menu screens.

The assets are original procedural artwork. They intentionally avoid copied game textures, logos, fonts, or interface files.
