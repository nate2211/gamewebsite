# Voxel Render Master Pack

The master pack contains 32 original 1024×1024 JPEG source textures. It is intentionally separate from the runtime 16×16 procedural block texture cache.

This arrangement gives the project:

1. Crisp, low-memory block faces during gameplay.
2. Large source artwork for future texture-pack selection, promotional renders, menus, and modding.
3. A larger archive containing useful production content rather than repeated or empty padding.

Each asset directory includes `manifest.json` with an ID, filename, material kind, resolution, and intended purpose.
